## 📖 Overview

**Rumah Amal Raudhatul Jannah** is a corporate-style informational website developed for a registered children's welfare home in Banting, Selangor. The site serves as a digital presence to introduce the organization to potential donors, volunteers, and visitors.

> ⚠️ **Disclaimer:** This website was developed for **educational purposes** as part of the IMS458 Web Design and Development course. It is not the official website of Rumah Amal Raudhatul Jannah.

---

## ✨ Features

| Feature | Description |
| :--- | :--- |
| 🏠 **Home Page** | Welcoming introduction with core values and quick navigation |
| 📋 **About Page** | Organization background, mission, vision, leadership structure, and commitment |
| 🎯 **Activities Page** | Program categories, daily routine timeline, and image gallery |
| 🤲 **Donation Page** | Support categories, instructions, and volunteer information |
| 📞 **Contact Page** | Address, phone, email, interactive map (hover to reveal), and contact form |
| 👤 **About Me Page** | Student/developer credentials with professional presentation |

---

## 🎨 Design Highlights

- **Color Palette:** Dark blue (`#1a4b6d`), teal green (`#2f7d62`), and warm gold (`#f5a623`)
- **Typography:** Playfair Display (serif) + Inter (sans-serif)
- **Loading Screen:** Smooth fade-in/out with branded spinner
- **Navigation:** Sticky menu with animated underline on hover
- **Responsive:** Fully optimized for desktop, tablet, and mobile devices
- **Interactive Map:** Hover to reveal Google Maps embed
